Cara menjalankan projek app.py

Buka terminal (new terminal di vs code)\

1. Masuk ke venv (virtual enviroment)
python -m venv venv
venv\Scripts\activate

2. Install library
pip install flask joblib numpy
pip install xgboost
pip install pandas
pip install scikit-learn or pip pip install sklearn
pip install openpyxl

3. Jalankan python
pyhton train_model.py
python app.py
